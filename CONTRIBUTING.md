# Contributing to Memu

## The Stack
* **OS:** Ubuntu 24.04 LTS
* **Hardware:** Intel N100 (x86-64)
* **Core:** Docker Compose, Python, Matrix, Immich

## Pull Requests
1.  Fork the repo.
2.  Test your changes on a clean install (or Docker Desktop).
3.  Ensure no secrets are committed.
4.  Submit PR!
